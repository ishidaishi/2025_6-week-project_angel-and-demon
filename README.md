# 2025_6 week project_angel and demon
 
## プロジェクト：6WP「天使と悪魔のリマインダー」（2025）

**期間**：6週間
**予算**：5万円  
**チーム構成**：工学系 3名・芸術系 2名（計5名）

本プロジェクトは、研究室で実施された短期開発企画「6WP（6 Weeks Project）」の一環として行われました。  



### 担当領域（私の役割）

- **組み込みソフトウェア開発全般**
  - Arduino等を用いたマイコン制御プログラムの実装
  - チームのハードウェア構成に合わせた動作確認とデバッグ
 
- **チームリーダー**

本プロジェクトでは、主に**デバイスの動作部分および組み込みソフトウェア**に関する開発を担当しました。

### プロジェクト概要
研究室のプロジェクトとして、工学系と芸術系の学生が協働し、6週間・5万円という制約の中でプロダクト開発に取り組みました。期間内に完成度の高いデモ発表を行い、**Maker Faire Tokyo**への出展を最終的な目標としました。

私はチームリーダー兼組み込み開発担当として、メンバーの強みを踏まえて役割分担を行い、週1回の全体ミーティングと日々のチャット連絡で進捗管理を徹底しました。特に、限られた期間で確実に成果を出すため、早期に試作品を完成させ、優先度の高い機能から改良を重ねるという開発方針を提案・実行しました。また、Maker Faireの出展には審査があるため、「魅せ方」も重視し、コンセプトが直感的に伝わるビデオや写真を制作しました。さらに、デモ本番で起こり得る不具合に対して事前に対策を施し、動作の信頼性を向上させるように意識しました。

その結果、期限内にプロダクトを完成させ、Maker Faire Tokyo 2025の審査も通過し、出展が実現しました。展示当日はポジティブな反応を多くいただき、人を笑顔にできたという手応えと進捗管理やリスク対策を含む事前準備を丁寧に積み重ねる重要性を感じました。

プロジェクトを通じて培ったプロジェクトマネジメント力は、限られた資源の中でチームをまとめ、着実に成果を出す力として活かせると考えています。
